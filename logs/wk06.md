## JavaScript essentials I learned

- **Variables (const / let)**: `const` is for values that don't get reassigned, `let` is for ones that change. I default to `const` and switch to `let` only when I need to.
- **Data types**: the basic ones are strings (text), numbers, and booleans (true/false), plus arrays and objects for grouping things.
- **Arrow functions**: a shorter way to write a function, like `() => { ... }`. I use them a lot for event listeners.
- **if / else**: runs one block when a condition is true and another when it's false, which is how the discount logic picks a tier.
