## JavaScript concepts

- **The DOM** is the page turned into objects that JavaScript can read and change. Each tag becomes a node I can grab and update.
- **Events** are things that happen on the page, like a click or typing, and I can run code when they happen.
- **querySelector vs getElementById**: `getElementById("x")` finds one element by its id. `querySelector(".x")` uses a CSS selector, so it's more flexible and can match classes or tags, and it returns the first match.
- **addEventListener** attaches a function to an element for a certain event, like `button.addEventListener("click", () => { ... })`. The function runs every time that event fires.
